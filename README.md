# Jedi Path
## Setting Up
In First, you need install:
- nodejs 10.x
- yarn
- lerna

After installing all entities run dependencies installation
```
lerna bootstrap
```
## Switch to Gateway
```
cd packages/swt-gateway
yarn run dev
```
# Sith Path

## Install Dep's
```
docker-compose run yarn
```
## Setup Gateway
```
docker-compose up swt-gateway
```

## For stage imitation just run
```
docker-compose up
```
